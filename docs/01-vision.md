# PERKABEM EVENT SYSTEM V2

## Vision

PES V2 adalah platform digital untuk mengelola seluruh kegiatan PERKABEM.

Sistem ini dirancang agar dapat digunakan untuk berbagai jenis event, bukan hanya HUT PERKABEM.

Target utama:

- Cepat digunakan oleh panitia
- Mudah dipelajari
- Mobile Friendly
- Realtime
- Scalable
- Professional

---

## Goals

1. Mengurangi pekerjaan manual panitia.
2. Digital Invitation.
3. QR Check-In.
4. Dashboard realtime.
5. Lucky Draw otomatis.
6. Reporting otomatis.
7. Multi Event.
8. Multi User.
